Augmented Reality

Virtual Reality

SOLID design principles

The SOLID principles do not only apply on software development but also when architecture a system

The principle of Least Astonishment

It proposes that a component of a system should behave in a way that most users will expect it to behave.

The principle of Least Effort

In principle, this means that if a solution to perform a task is found, there is no need to improve the existing method.

Principles of Economics

choose a path to add maximum value to the project, and no action should be delayed until the moment when it is absolutely required
